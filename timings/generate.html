Sentence Timing Generator

One-click, in-browser. Generates the JSON timing files that lock the highlighted sentence to the narration.

How this works

Your browser will download a small (~150 MB) speech model and use it to find when each sentence starts in every audio track. Word-level timestamps are matched against the book's exact sentence list (sentences.json) — no transcription guesswork.

This is a one-time job. When done, click "Download timings" — drop the resulting JSON files into book/timings/ and the highlighting locks to the voice automatically.

Heads up

This will run for 30–90 minutes depending on your machine. It downloads ~150 MB of model weights on first run (cached after that). Keep this tab open and your laptop awake. You can keep using other apps while it runs.

If a section fails, the rest still complete — you can re-run only the failed ones.
